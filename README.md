### 视图函数详解 1. upload() - 首页视图

- 职责 ：渲染上传页面作为系统首页
- 参数 ： request - Django HTTP 请求对象
- 返回 ：渲染后的 upload.html 模板 2. predict() - 异步预测视图 ⭐核心
  执行流程：

步骤 操作 说明 1 request.method == 'POST' 验证是否为 POST 请求 2 ImageUploadForm 验证 检查文件类型是否合法 3 FileSystemStorage.save() 保存图片到 media 目录 4 asyncio.run\_in\_executor() 异步执行 模型预测 5 render(result.html) 返回识别结果页面

### 关键技术特性 ✅ 异步处理（创新点）

- 使用 asyncio.run\_in\_executor() 将模型预测任务提交到线程池
- 避免阻塞 Django 主线程，支持并发请求 ✅ 文件存储
- 自动保存到 MEDIA\_ROOT （即 media/ 目录）
- 返回文件访问 URL 供前端展示 ✅ 表单验证
- 验证文件类型（仅允许 jpg/jpeg/png/bmp/gif）
- 验证文件大小和完整性

### 总结

views.py 承担以下职责：

1. 接收请求 ：处理用户的图片上传请求
2. 验证数据 ：使用表单验证确保文件合法性
3. 调用业务逻辑 ：通过 utils.py 执行模型预测
4. 返回响应 ：渲染模板返回识别结果
   采用 异步视图 设计，提升了并发处理能力，避免模型加载和预测过程阻塞主程。

## **"tests.py":**

这是一个 Django 单元测试文件 ，用于自动化测试项目的核心功能，确保代码质量和功能正确性。

## tests.py 功能说明

### 测试类结构

测试类 测试范围 测试用例数 ImageUploadFormTest 图片上传表单验证 4个 ModelLoadingTest 模型加载功能 3个 PredictImageTest 图片预测功能 7个

### 具体测试内容

1. 表单验证测试 ( ImageUploadFormTest )

- 测试所有允许的图片格式（jpg/jpeg/png/bmp/gif）能通过验证
- 测试不允许的格式（txt/pdf/doc/zip/exe）被正确拒绝
- 测试空文件被拒绝
- 测试缺少图片字段时表单无效

1. 模型加载测试 ( ModelLoadingTest )

- 测试模型加载函数存在
- 测试类别名称列表定义正确（5个类别）
- 测试中文类别名称字典完整

1. 预测功能测试 ( PredictImageTest )

- 测试预测函数存在
- 测试返回字典格式正确，包含所有必需字段
- 测试置信度在 0-1 范围内
- 测试标签代码在允许的类别列表中
- 测试预测结果只能是"是"或"不是"
- 测试返回完整概率数组（5个元素）
- 测试概率数组之和接近1

### 运行测试

## **深度学习部分（ResNet50、MobileNet\_V2）**

这两个文件是五谷识别系统的 模型训练脚本 ，用于训练 MobileNetV2 和 ResNet50 两个深度学习模型，采用 两阶段迁移学习 策略。

### 文件对比

文件 模型 解冻层数 最终模型文件 特点 train-MobileNetV2.py MobileNetV2 5层 models/mobilenetv2\_grains.h5 轻量、快速 train\_resnet.py ResNet50 10层 models/resnet50\_grains.h5 精度高、较大

### 训练流程（两阶段迁移学习） 第一阶段：冻结预训练层，只训练分类头

分类头结构：
第二阶段：解冻顶层卷积，小学习率微调

### 关键技术特性 ✅ 数据增强 ✅ 回调函数

回调 作用 参数 EarlyStopping 早停防止过拟合 patience=10（第一阶段）/8（第二阶段） ModelCheckpoint 保存最佳模型 monitor='val\_accuracy', save\_best\_only=True ReduceLROnPlateau 学习率衰减 factor=0.5, patience=5

### 模型架构对比

对比项 MobileNetV2 ResNet50 参数量 \~2.2M \~25.6M 模型大小 \~14MB \~95MB 推理速度 快 较慢 精度 中等 较高 解冻层数 5层（保守） 10层

### 运行方式

```
# 训练 MobileNetV2
python train-MobileNetV2.py

# 训练 ResNet50
python train_resnet.py
```

### 输出文件

```
models/
├── mobilenetv2_grains.h5          
# MobileNetV2最终模型
├── mobilenetv2_grains_phase1_best.
h5  # 第一阶段最佳模型
├── resnet50_grains.h5             
# ResNet50最终模型
└── resnet50_grains_phase1_best.
h5    # 第一阶段最佳模型
```

### 设计亮点

1. 两阶段训练 ：先训练分类头，再微调顶层卷积，保护预训练特征
2. 小样本安全微调 ：仅解冻少量层 + 极小学习率，避免过拟合
3. 正则化策略 ：L2正则化 + Dropout + BatchNormalization 三重防过拟合
4. 早停机制 ：监控验证集损失，自动停止并恢复最佳权重
5. 学习率衰减 ：验证集损失不再下降时自动降低学习率
